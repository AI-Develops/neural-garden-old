# Neural Garden

A living static website created using pure HTML5 and modern CSS3.

No JavaScript. No frameworks. No dependencies.

## Concept

Neural Garden represents ideas as living entities in a digital ecosystem.

It demonstrates modern CSS features including:

- container queries
- scroll-driven animations
- color-mix()
- clamp()
- accent-color
- backdrop-filter
- CSS layers
- relational selectors (:has)
